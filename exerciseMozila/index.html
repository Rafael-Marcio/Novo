O Firefox é um navegador da web de código aberto desenvolvido pela Mozilla. Ele é conhecido por sua ênfase na privacidade, segurança e personalização. O Firefox é utilizado por milhões de pessoas em todo o mundo.A Mozilla tentou entrar no mercado de sistemas operacionais móveis com o Firefox OS, um sistema operacional de código aberto para smartphones. No entanto, em 2016, a Mozilla anunciou que estava encerrando o desenvolvimento do Firefox OS para smartphones devido a desafios competitivos.
